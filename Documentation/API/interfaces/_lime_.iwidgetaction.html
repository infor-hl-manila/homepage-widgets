Optional isEnabledNotConfigured

isEnabledNotConfigured: boolean
  • Defined in lime.ts:375

Gets or sets a value that indicates if the action should be available even if the widget is not configured.

Actions are normally not available when a widget that requires configuration is not configured. This property can be used to override this behavior for specific actions that can still be used when the widget is not configured.

The default value is false.

1.0.34
