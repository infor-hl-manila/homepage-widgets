Static format

  • format(value: any, options?: INumberFormatOptions): string
    • Defined in lime.ts:3654

    Formats a number or string using default or specified formatting options.

    This function currently only supports formatting using a specified decimal separator. The value to format is expected to be either a number or a string containing a number where any decimal separator is dot (.). If the value is a string it may not contain any thousand separators or other formatting characters.

    Parameters

    • value: any

      The value to format which can be a number or a number in string format.

    • Optional options: INumberFormatOptions

      Optional formatting options.

    Returns string
